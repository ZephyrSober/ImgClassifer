# ImgClassifer
This is a image classifer based on MindSpore. It basicly identify [cats] and [dogs]. More labels will be supported in the future.

# TIPS
Before you run these codes, download dataset file into path "./MindSporePetClassification/code/dataset".
If you want to use converter tools, put it at "./MindSporePetClassification/converter".
Raw dataset(.zip file) should be put at "./MindSporePetClassification/*.zip".
